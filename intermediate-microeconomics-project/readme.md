# Description
1. desmos-links.js
   - list of links for each graph.  This script places a link to the graph in the graph page. For example, line 37 of 4-axis-Cost-Benefit.html
  
2. explanation.js
   - List to html code/text snippets that can be inserted in the html files.
   - At the moment there are four ways it is applied
      1. List of headers for each graph, e.g. 4-axis-Cost-Benefit-header, loaded in line 27 of 4-axis-Cost-Benefit.html
      2. List of explanation for each graph, e.g. 4-axis-Cost-Benefit, loaded in line 65 of 4-axis-Cost-Benefit.html
      3. See week1/Quiz.html Line 38 load W1Quiz (questions)
      4. See week1/Quiz.html  Line 65 loads W1Quizans (Answers)
    - note, two definitions of const so far, "const POLICY_EXPLANATIONS = {" "const Utility_Types = {"  to add clickable text
4. index.html
   - Main page
5. instructor.html
   - Instructor meain page
6. video.js
   - list of videos, e.g loaded in line 59 of 4-axis-Cost-Benefit.html
